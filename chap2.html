Chapter 2: Number System, Conversion and Boolean Logic

2.1 Number System and Conversion

2.1.1 Decimal, Binary, Octal, Hexadecimal Number System & Conversion

The number system is the foundation of all computing systems. The four main number systems used in computing are:

  • Decimal (Base-10): This system uses 10 digits (0-9). It is the most common system used in everyday life.
  • Binary (Base-2): The binary system uses two digits, 0 and 1. Computers process data in binary.
  • Octal (Base-8): The octal system uses 8 digits (0-7). It is commonly used as shorthand for binary numbers.
  • Hexadecimal (Base-16): The hexadecimal system uses 16 symbols (0-9 and A-F), representing values from 0 to 15. It simplifies representing large binary numbers.

Conversion between these systems can be done using division, multiplication, or bitwise operations. For example, to convert binary to decimal, you multiply each binary digit by 2 raised to the power of its position.

2.1.2 Calculation in Binary: Addition, Subtraction

Binary addition and subtraction are similar to decimal operations but follow simple rules based on binary digits.

Binary Addition:

For binary addition, we use the following rules:

  • 0 + 0 = 0
  • 1 + 0 = 1
  • 1 + 1 = 10 (which means carry over 1 to the next higher bit)
            1010
          + 1101
          ------
           10111

Binary Subtraction:

For binary subtraction, we use the following rules:

  • 0 - 0 = 0
  • 1 - 0 = 1
  • 1 - 1 = 0
  • 0 - 1 requires borrowing from the next higher bit
            1010
          - 0111
          ------
            0011

2.1.3 One’s and Two’s Complement Methods of Binary Subtraction

The 1's and 2's complement methods are used to perform binary subtraction. They allow subtraction to be performed using binary addition.

1's Complement:

The 1's complement of a binary number is obtained by flipping all bits (changing 0s to 1s and 1s to 0s).

2's Complement:

The 2's complement of a binary number is obtained by adding 1 to the 1's complement of the number. This method is commonly used in computing to represent negative numbers.

Example (Board Exam Question)

Q) Subtract (1000)2 from (111)2 using 1's and 2's complement.

Using 1's Complement:

First calculate the 1's complement of (1000)2: (0111)2.

Adding (0111)2 with (111)2, we get (1110)2.

There is no extra bit. Now calculate the 1's complement of (1110)2, which is (0001)2. Hence, the result is -(0001)2.

Using 2's Complement:

First calculate the 2's complement of (1000)2: (0111)2 + (1)2 = (1000)2.

Adding (1000)2 with (111)2, we get (1111)2.

There is no extra bit. Now calculate the 2's complement of (1111)2: (0000)2 + (1)2 = (0001)2. Hence, the result is -(0001)2.

2.2 Logic Function and Boolean Algebra

2.2.1 Introduction to Boolean Algebra

Boolean Algebra is the branch of mathematics that deals with binary variables and logical operations. It is used to simplify logical expressions, which is essential for digital circuit design.

2.2.2 Introduction to Boolean Values, Truth Table, Boolean Expression and Boolean Function

In Boolean algebra, values are restricted to two states: true (1) and false (0). A Boolean expression is a combination of Boolean values, operators (AND, OR, NOT), and variables. The truth table shows all possible combinations of input values and the corresponding output for a Boolean function.

2.2.3 Logic Gates – AND, OR, NOT, NAND, NOR, XOR, and XNOR

Logic gates are the basic building blocks of digital circuits. These gates perform specific logical functions on one or more binary inputs to produce a binary output.

AND Gate (A . B) OR Gate (A + B) NOT Gate (~A)

0 + 0 = 0

2.2.4 Laws of Boolean Algebra – Boolean Identities

Boolean algebra has several laws that simplify expressions and solve logical problems. Some of the fundamental laws include:

  • Identity Law: A + 0 = A, A . 1 = A
  • Complement Law: A + A' = 1, A . A' = 0
  • Commutative Law: A + B = B + A, A . B = B . A
  • Associative Law: (A + B) + C = A + (B + C), (A . B) . C = A . (B . C)
  • Distributive Law: A . (B + C) = (A . B) + (A . C), A + (B . C) = (A + B) . (A + C)

2.2.5 Statement and Verification of Laws of Boolean Algebra Using Truth Table

To verify the laws of Boolean algebra, we can use truth tables. A truth table is a table that lists all possible combinations of inputs and shows the corresponding output for a given Boolean expression.
