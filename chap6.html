Chapter 6: Web Technology I

Web technology encompasses the tools and techniques used to build and manage websites. This chapter introduces the fundamentals of web development, including HTML, CSS, and related concepts.

6.1 Introduction: Web Development

Web development is the process of building websites and web applications. It involves front-end (client-side) development, which handles the user interface, and back-end (server-side) development, which deals with databases and server-side logic.

6.2 Web Browsers and Search Engines

Web Browsers: A web browser is a software application used to access and view websites. Popular browsers include Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge.

Search Engines: Search engines like Google, Bing, and Yahoo help users find relevant information on the internet by indexing websites and returning results based on search queries.

6.3 Overview of Various Internet & Web Technologies

The internet and web technologies work together to enable communication and interaction online. Web technologies include:

  • HTML: The backbone of web content, used for structuring webpages.
  • CSS: Styling web pages with design elements like colors, fonts, and layout.
  • JavaScript: A programming language for adding interactivity and dynamic content.
  • Web Servers: Servers that host and serve web pages to users.
  • Databases: Used for storing and retrieving data for websites.

6.4 HTML: The Language of the Web

HTML (HyperText Markup Language) is the standard language used to create and design web pages. HTML provides structure to content, using elements such as headings, paragraphs, and links.

6.4.1 Objectives

By learning HTML, you can build static web pages and define the basic layout of content on the web.

6.4.2 Structure of HTML

An HTML document consists of a declaration (<!DOCTYPE>), opening and closing HTML tags (<html>), and various child elements like the <head> and <body> sections.

6.4.3 Published and Hosting

Publishing a website involves uploading files to a web server using FTP (File Transfer Protocol). Hosting services are used to store website files and make them accessible to users on the internet.

6.4.4 HTML Tags vs. Attributes

HTML tags define elements like headings and paragraphs, while attributes provide additional information about those elements. For example, <a href="link"> uses the href attribute to define a hyperlink's destination.

6.4.5 Basic Tags of HTML

  • HTML Tag: Defines the entire webpage.
  • HEAD Tag: Contains metadata about the page (like title and styles).
  • TITLE Tag: Specifies the title of the webpage displayed in the browser tab.
  • BODY Tag: Contains the main content of the page.
  • Setting the Forecolor and Background Color: <body bgcolor="color">
  • Background Image: <body background="image.jpg">
  • Background Sound: <body sound="sound.wav">

6.4.6 Heading Tag (H1 to H6) and Attributes

Heading tags define the hierarchy of content, with <h1> being the largest and most important, down to <h6>.

6.4.7 FONT Tag and Attributes

The <font> tag is used to set the font style, size, and color. For example, <font size="3"> changes the font size to level 3.

6.4.8 Paragraph Formatting (P)

The <p> tag is used for creating paragraphs. You can also align text within paragraphs using the align attribute.

6.4.9 Break Line (BR)

The <br> tag inserts a line break, forcing content to the next line without starting a new paragraph.

6.4.10 Comment in HTML

Comments in HTML are added with <!-- comment text -->. These comments are not displayed in the browser.

6.4.11 Formatting Text

HTML provides tags for formatting text, such as <b> for bold, <i> for italics, <u> for underline, and others like <mark> and <blockquote>.

6.4.12 Ordered List (OL)

Use <ol> to create ordered lists and <li> for list items. You can specify the list style (numbers, letters, Roman numerals) with the type attribute.

6.4.13 Unordered List (UL)

Unordered lists are created using the <ul> tag, with list items defined by <li>. You can change the bullet type using attributes like disc, circle, and square.

6.4.14 Address Tag

The <address> tag is used to define contact information, typically displayed in italics.

Creating Links

Links are created using the anchor <a> tag. You can link to other documents, pages, or sections within the same document.

6.4.15 Tables: Creating Tables using TH, TR, and TD Tags

HTML tables are created using the <table> tag, with rows defined by <tr>, header cells by <th>, and data cells by <td>.

6.4.16 Forms: Creating Forms

Forms are created using the <form> tag. Input elements like textboxes, radio buttons, checkboxes, text areas, and buttons are added using respective tags such as <input>.

6.4.17 HTML 5 Elements

HTML5 introduces elements like <audio>, <video>, <canvas>, and <svg> for multimedia content and graphics.

6.4.18 HTML 5 Graphics using Canvas and SVG Tags

Canvas and SVG tags allow developers to create dynamic graphics directly in the browser. Canvas is used for drawing shapes and images, while SVG is used for scalable vector graphics.

6.4.19 Concept of Domain Name and Web Hosting

A domain name is the address of a website, while web hosting is the service that stores and makes website files accessible online.

6.5 Cascading Style Sheets (CSS)

6.5.1 Introduction to CSS

CSS is used for styling and designing web pages. It controls layout, colors, fonts, and other presentation aspects of a page.

6.5.2 Inline CSS

Inline CSS is used to style individual elements with the style attribute directly in the HTML tag.

6.5.3 Embedded CSS

Embedded CSS is written within the <style> tag in the <head> section of the HTML document.

6.5.4 External CSS

External CSS is stored in separate .css files, allowing for consistent styling across multiple pages of a website.
