Chapter 3: Computer Software and Operating System

Software refers to the collection of programs that tell a computer how to perform tasks, while the operating system (OS) is a critical program that manages the computer's hardware and software resources.

3.1 Concept of Software

3.1.1 Definition of Software

Software is a collection of data or instructions that tell a computer how to perform specific tasks. It can be categorized into different types based on its functionality, such as system software, application software, and utility software. Software makes the hardware of a computer operational by providing a user interface and facilitating the execution of complex operations.

3.1.2 Categories of Software

Software can be broadly classified into the following categories:

  • System Software: Includes the operating system (OS), device drivers, and utilities that manage and control hardware and other resources. The OS serves as the interface between the user and the computer hardware.
  • Utility Software: These are programs designed to perform a specific task related to the management of system resources. Examples include antivirus software, file management tools, and disk cleanup utilities.
  • Application Software: These are programs designed to perform specific tasks for the user, such as word processing (e.g., Microsoft Word), spreadsheets (e.g., Excel), or web browsers (e.g., Google Chrome).
  • Web-Based Software: These applications run on web servers and can be accessed through web browsers. Examples include Google Docs, online banking platforms, and social media websites.
  • Mobile Apps: These are software applications designed for mobile devices like smartphones and tablets. Popular examples include WhatsApp, Instagram, and mobile banking apps.

3.2 Concept of Operating System

3.2.1 Introduction to Operating System

The operating system (OS) is system software that manages hardware resources and provides services for computer programs. It serves as an intermediary between users and the computer hardware. Popular OS examples include Windows, macOS, and Linux. An OS ensures that software programs function correctly by allocating resources such as memory, CPU time, and input/output devices.

3.2.2 Role of Operating System

The operating system performs several critical functions, including:

  • Resource Management: Manages hardware resources like the CPU, memory, storage devices, and peripheral devices.
  • Task Management: Allocates tasks to the CPU and schedules them for execution. It also ensures that multiple applications can run concurrently without interfering with each other.
  • Security and Access Control: Protects the system from unauthorized access and ensures data privacy and integrity through user authentication and authorization systems.
  • User Interface: Provides a user interface (UI), either command-line (CLI) or graphical (GUI), for interacting with the system and applications.

3.2.3 Functions of an Operating System

The primary functions of an operating system include:

  • Process Management: Controls the execution of processes, allocates CPU time, and ensures that processes do not interfere with each other.
  • Memory Management: Handles the allocation and deallocation of memory to processes and ensures efficient memory use.
  • File Management: Provides an organized way of storing, retrieving, and managing files and directories on storage devices.
  • Device Management: Manages hardware devices like printers, monitors, and disk drives by providing drivers and ensuring proper communication between the hardware and software.
  • Security Management: Provides mechanisms to secure the system against unauthorized access, including encryption, firewalls, and user access control.

3.2.4 Operating System Terminology

The following terms are commonly used in the context of operating systems:

  • Multiprogramming: The ability of an OS to execute multiple programs concurrently by managing their execution in parallel.
  • Multitasking: Refers to the OSâ€™s ability to manage multiple tasks or processes at once by allocating time slices to each task.
  • Multiprocessing: Involves using multiple processors to perform tasks simultaneously, improving system performance.
  • Distributed Systems: A system in which multiple computers work together to perform a task, often over a network. Each machine has its own OS, and they communicate to achieve a common goal.

3.3 Windows Operating System

3.3.1 Introduction to GUI-based Operating System and Its Features

The Windows operating system is a GUI-based system, meaning it provides a graphical user interface (GUI) for users to interact with the system through icons, buttons, and windows. Key features include:

  • Ease of Use: The GUI makes it easier for users to perform tasks like opening files, running programs, and managing settings through visual elements.
  • Multi-tasking: Windows allows users to run multiple applications simultaneously, improving productivity and multitasking efficiency.
  • System Configuration: Windows provides several tools like the Control Panel and Settings to configure and manage system settings.

3.3.2 Working in Desktop Application and Window Environment

The desktop environment in Windows includes a Start Menu, Taskbar, and Desktop, which help users access programs, manage files, and customize settings. Each application runs in its own window, and users can switch between multiple windows easily.

3.3.3 Manage Files and Folders with File Explorer

File Explorer is the default file management tool in Windows. It allows users to navigate through files and folders, organize them, and perform tasks such as copying, moving, and deleting files.

3.3.4 Customize the Start Screen and Desktop

Windows allows users to personalize their experience by customizing the Start Menu and desktop, such as changing the wallpaper, rearranging icons, or adding shortcuts to frequently used applications.

3.3.5 Installing and Removing Devices

Users can install and remove hardware devices through the Device Manager in Windows. When a device is connected, the OS automatically installs the necessary drivers. Devices can also be removed or uninstalled through the same interface.

3.3.6 Manage Passwords and Privacy Levels

Windows provides security features like password management, user account controls, and privacy settings to protect personal data and prevent unauthorized access.

3.3.7 Use of Control Panel, System Tools, and Accessories

The Control Panel and System Tools allow users to manage system configurations, access administrative tools, and troubleshoot problems. Accessories like Calculator, Paint, and Notepad are also part of the Windows ecosystem for basic utility needs.

3.4 Open Sources and Mobile Operating System

3.4.1 Concept of Open Source Operating System

An open-source operating system is one where the source code is available for anyone to view, modify, and distribute. Linux is one of the most popular open-source operating systems, allowing developers to contribute and create customized versions of the OS.

3.4.2 Introduction to Linux and UNIX

Linux is an open-source, Unix-like operating system that is widely used for servers, desktops, and embedded systems. UNIX, a predecessor to Linux, was originally developed in the 1970s and has influenced many modern operating systems.

3.4.3 Linux Distributions

Linux distributions (distros) are various versions of the Linux OS that include the kernel and additional software packages. Examples include Ubuntu, Fedora, and Debian, each designed for different use cases and user needs.

3.4.4 Concept of Mobile Operating System

Mobile operating systems (OS) are designed specifically for mobile devices such as smartphones and tablets. They handle touch inputs, wireless communication, and mobile-specific features like battery management.

3.4.5 Types of Mobile Operating System

The main mobile operating systems include:

  • Android: An open-source mobile OS developed by Google, widely used on smartphones and tablets.
  • iOS: Apple's proprietary mobile OS, found on iPhones, iPads, and iPods.
  • Windows Phone: Microsoft's mobile OS, which was discontinued but was once a popular competitor to Android and iOS.
